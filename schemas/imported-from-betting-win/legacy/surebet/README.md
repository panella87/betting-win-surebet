# Imported legacy surebet schema drafts

This directory contains historical SQL/schema drafts copied from the original `betting-win` repository.

```text
legacy_surebet_import_status=imported_and_rehomed
active_authority=no
active_schema_authority=no
production_schema_owner=none
```

These files are not production database migrations. They are retained as historical design references for surebet research only.

`betting-win` remains the owner of provider/history schemas. `betting-win-surebet` may own only surebet strategy state and derived paper/backtest artifacts.
